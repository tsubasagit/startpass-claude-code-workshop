## 概要

[StartPass](https://startpass.jp/) と [AppTalentHub](https://apptalenthub.co.jp) が共催する、ファウンダー向け Claude Code ハンズオンワークショップの研修資料サイトです。

2 時間のセッションを通じて、AI を活用した事業計画の策定から MVP のデプロイまでを体験できます。

> **StartPass の会員になると、こうした限定ワークショップにフルで参加できます。**
> 詳しくは [StartPass 公式サイト](https://startpass.jp/) をご覧ください。

### ワークショップの流れ（6 フェーズ）

| Phase | 内容 |
|-------|------|
| 1 | イントロダクション — Claude Code とは |
| 2 | Claude Code セットアップ |
| 3 | AI で事業アイデアを磨く |
| 4 | MVP のアーキテクチャ設計 |
| 5 | MVP レビュー＆改善 |
| 6 | まとめ＆ Q&A |

## ページ構成

- **トップ** — ワークショップ概要・タイムライン
- **事前準備** (`/setup`) — Node.js / Claude Code / GitHub のセットアップ手順
- **5ヵ年事業計画** (`/business-plan`) — AI 活用の事業計画策定ガイド・Plan Mode ルール
- **MVP 構築** (`/mvp`) — マイルストーン・成果物・成功指標
- **学習リソース** (`/resources`) — 成果物一覧・参考リンク
